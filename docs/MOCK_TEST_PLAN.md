# 📋 깡쌤토익 PC 전용 Full 모의고사(Set 1) 구축 기획서 (최종안)

## 1. 개요 (Overview)
본 문서는 깡쌤토익의 실전 감각 극대화를 위한 200문항 Full Mock Test의 구현 계획입니다.

## 2. 레이아웃 및 환경 (UX/UI)
- **환경**: PC 전체 화면 모드 (사이드바 제거).
- **2단 북클릿**: [왼쪽 페이지(50%) : 오른쪽 페이지(50%)] 분할 및 중앙 가이드라인.
- **구분선**: 모든 2분할/4분할 컬럼 사이에 **세로 실선**을 삽입하여 가독성 강화.
- **상단 헤더**: [현재 파트 | 페이지 정보 | 타이머] 단일 행 배치.
- **페이지 전환**: [이전 장 / 다음 장] 버튼을 통한 2페이지 단위 이동.

## 3. 파트별 상세 구성 (Layout Detail)

| 파트 | 흐름 및 레이아웃 상세 |
| :--- | :--- |
| **Part 1** | 1장(좌): Directions / 1장(우)~2장: 페이지당 사진 2장(상/하) + 원형 마킹 |
| **Part 2** | 페이지 내 2분할 컬럼 (좌 7-19번 | 구분선 | 우 20-31번) |
| **Part 3~4** | 페이지당 12문학 (좌 6 | 구분선 | 우 6). 시각자료는 텍스트 상단 배치 |
| **Part 5** | **새 장(Spread) 시작**. 페이지당 10문제 (좌 5 | 구분선 | 우 5). 총 3페이지 |
| **Part 6** | **P5 종료 옆 페이지(우) 바로 시작**. 지문 상단 + 하단 2분할 (좌 2 | 구분선 | 우 2) |
| **Part 7 (싱글)** | **P6 종료 옆 페이지(우) 바로 시작**. 지문 상단 + 하단 2분할 분산 배치<br/>- 2문제: 좌1/우1, 3문제: 좌2/우1, 4문제: 좌2/우2, 5문제: 좌3/우2<br/>- **마지막 지문**: 왼쪽 페이지(지문) / 오른쪽 페이지(문제 2컬럼) 배치 |
| **Part 7 (멀티)** | 기존 서비스의 상하/좌우 다중 패널 레이아웃 적용 유지 |

## 4. 로직 및 데이터 (Core Logic)
- **데이터**: 각 파트별 **TEST 9회차** 전량 사용.
- **LC 오디오**: 45분간 전체 재생, **일시정지/정지 불가** (실전 모드).
- **RC 타이머**: RC 진입 시 **75분 카운트다운**. 종료 시 모든 입력 차단 및 자동 제출.
- **마킹**: 문제 옆 **원형 버튼(A,B,C,D)** 클릭 시 즉시 데이터 저장.

## 5. 구현 우선순위
1.  LC 섹션 (P1~P4) 레이아웃 및 오디오 통합.
2.  RC 섹션 (P5~P7 싱글) 가변 컬럼 및 심리스 흐름 구현.
3.  RC 타이머 및 자동 종료 시스템.
