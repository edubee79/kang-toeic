## 🎯 빠른 참조

### 회원가입 반 vs 목표 점수
- **회원가입 반**: 650반/800반/900반 (고정)
- **목표 점수**: 나의 학습방에서 자유롭게 입력/수정 가능

### RC/LC 분배 공식
```typescript
// LC = RC + 50 조건
const rc = Math.round((totalTarget - 50) / 2);
const lc = totalTarget - rc;
```

### TOEIC 점수 분포표 (핵심)
**RC (100문제)**
- 350점 → 75개 정답 필요 (75%)
- 400점 → 85개 정답 필요 (85%)
- 450점 → 92개 정답 필요 (92%)

**LC (100문제)**
- 400점 → 78개 정답 필요 (78%)
- 450점 → 85개 정답 필요 (85%)
- 495점 → 95개 정답 필요 (95%)

### 어휘 학습 3단계
1. **Sort**: 2단계 확인 ("알아요" → 뒤집기 → "진짜 알아요/몰라요")
2. **Learn**: 복습 풀 + Fisher-Yates 셔플
3. **Test**: 4지선다 + 3초 타이머 + 80% 합격

### SRS 복습 간격
- 1일 → 3일 → 7일 → 14일 → 30일
- 오답 시 1일로 리셋
